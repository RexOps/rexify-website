If you want to make changes and test these changes on your workstation follow these steps:

=item *

Install Perl modules: C<cpanm DateTime IO::All Mojolicious Mojolicious::Plugin::RenderFile>

=item *

Start a local webserver: C<cd rexify-website/html && hypnotoad website.pl>

=item *

Open L<http://localhost:8080> in your favorite browser
